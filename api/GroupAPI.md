RainyBot的群组API类，所有的群组相关API将直接或间接继承此类  
  
## 描述  
  
这是RainyBot的群组API类，所有的群组相关API将直接或间接继承此类   
继承此类的API包括但不限于群组/群组配置/群列表/群成员/群成员列表等类
